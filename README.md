## 〔 Установка модулей 〕

В Heroku userbot выполни команду:

```
.dlm https://raw.githubusercontent.com/justidev-heroku/justi-modules/main/имя_модуля.py
```

или добавь репозиторий через:

```
.addrepo https://github.com/justidev-heroku/justi-modules
```
